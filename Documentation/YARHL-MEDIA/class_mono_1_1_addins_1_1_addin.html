◆ Enabled

bool Mono.Addins.Addin.Enabled
getset

Gets or sets the enabled status of the add-in.

This property can be used to enable or disable an add-in. The enabled status of an add-in is stored in the add-in registry, so when an add-in is disabled, it will be disabled for all applications sharing the same registry. When an add-in is enabled or disabled, the extension points currently loaded in memory will be properly updated to include or exclude extensions from the add-in.
