string Mono.Addins.Description.ExtensionNodeDescription.GetParentPath ( )

Gets the extension path under which this node is registered

Returns
The parent path.

For example, if the id of the node is 'ThisNode', and the node is a child of another node with id 'ParentNode', and that parent node is defined in an extension with the path '/Core/MainExtension', then the parent path is 'Core/MainExtension/ParentNode'.
