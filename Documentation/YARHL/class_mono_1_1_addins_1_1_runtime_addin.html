object Mono.Addins.RuntimeAddin.CreateInstance ( string  typeName)

Creates an instance of a type defined in the add-in

Parameters
typeNameName of the type.
Returns
A new instance of the type

The type will be looked up in the assemblies that implement the add-in, and recursively in all add-ins on which it depends.

This method throws an InvalidOperationException if the type can't be found.

The specified type must have a default constructor.
